## 📌 About the Project

**FACEFORGE** is an AI-powered system that generates realistic human faces using **Generative Adversarial Networks (GANs)**.

The project enhances traditional GAN architectures by improving both:

* 🧠 **Generator** → Produces realistic faces
* 🔍 **Discriminator** → Detects real vs fake images

It leverages the **CelebA dataset** to learn diverse facial features and generate high-quality synthetic images.
